\section*{Notation and Operator Standards}
\label{sec:gloss_notation}

To maintain clarity across diverse domains, the following custom operators are utilized:

egin{description}
    \item[	extbf{The Anchor Operator} ($nchor{}$)] \hfill \
    	extit{Designation: Structural Invariant / Fixed Point ($C_{fix}$)} \
    The operator $nchor{}$ denotes a coordinate or value within a manifold that remains constant while the surrounding domain undergoes transformation. It serves as an unchanging reference point for the operation.
    \[0.5em]
    	extbf{Axiom:} For any transformation map $T: S 	o S$, if an element $x$ is bound by $nchor{}$ (denoted $nchor{x}$), then $T(x) = x$.

    \item[	extbf{The Parity Operator} ($\parity$)] \hfill \
    	extit{Designation: Symmetry Correspondence / Chirality} \
    The operator $\parity$ defines the inversion signature (handedness) of a state relative to the Locus. It determines how a value responds to spatial reflection.
    \[0.5em]
    	extbf{States:}
    egin{itemize}
        \item[$(+)$] 	extbf{Symmetric:} The system is Self-Similar (Identity). $f(x) = f(-x)$.
        \item[$(-)$] 	extbf{Anti-Symmetric:} The system is Self-Opposite (Inversion). $f(x) = -f(-x)$.
        \item[$(\equiv)$] 	extbf{Equilibrium:} The system is Perfectly Reciprocal (Unitary Balance).
    \end{itemize}
\end{description}
